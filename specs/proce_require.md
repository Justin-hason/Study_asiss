
# Spec 1：智能学习助手系统 — 功能规格说明书（基于流程需求）

## 文档信息
- 版本：V1.0
- 目标用户：开发者、测试工程师、项目经理
- 描述：以13步核心流程为主线，定义系统必须实现的功能点与行为约束。

## 1. 范围与定义
本系统为基于RAG+私有知识库的智能学习平台，覆盖资料上传、知识库管理、智能问答、学习规划、错题复盘与数据统计。功能规格严格遵照以下13步流程，每一步对应一组功能需求。

## 2. 功能需求（按流程步骤）

### 步骤1：注册、登录与权限设置
| 需求编号 | 功能描述 |
|----------|----------|
| F1.1 | 支持手机号、邮箱注册，及微信/SSO第三方登录。 |
| F1.2 | 注册时引导选择学习方向标签，系统据此初始化推荐模板。 |
| F1.3 | 提供双因素认证（可选）与会话管理，检测异地登录。 |
| F1.4 | 角色分为普通用户、机构管理员、系统管理员，权限矩阵可配。 |

### 步骤2：资料上传与向量化入库
| 需求编号 | 功能描述 |
|----------|----------|
| F2.1 | 支持上传格式：PDF、Word、Markdown、TXT、PPT、图片（OCR）、扫描件PDF（内嵌OCR）。 |
| F2.2 | 上传时自动执行文本提取、清洗（去页眉页脚/水印），保留表格与LaTeX公式。 |
| F2.3 | 自动解析文档结构（标题层级、段落、图片），生成文档大纲树。 |
| F2.4 | 智能分块：语义段落切分，滑动窗口重叠，单块≤512 tokens。 |
| F2.5 | 提取元数据：文件名、上传时间、页数、语言。 |
| F2.6 | 生成向量并存入向量数据库，上传进度实时反馈，失败提示原因。 |

### 步骤3：知识库结构化管理
| 需求编号 | 功能描述 |
|----------|----------|
| F3.1 | 用户可创建多级文件夹，支持拖拽移动、批量归类。 |
| F3.2 | 自定义标签（颜色、快捷键），系统基于内容自动推荐标签。 |
| F3.3 | 文档级权限：私有、指定好友可见、机构内公开、链接分享（可设密码/有效期）。 |
| F3.4 | 目录视图支持树形/列表，按时间/大小/类型排序，可全文搜索文件名。 |
| F3.5 | 文档版本管理：保留历史版本，支持回溯对比。 |

### 步骤4：问题输入与语义解析
| 需求编号 | 功能描述 |
|----------|----------|
| F4.1 | 全局对话式提问框，支持文字、语音输入（ASR）、拍照搜题（数学公式OCR）。 |
| F4.2 | 意图识别：区分学习问答、搜题、材料生成、操作指令。 |
| F4.3 | 查询重写与扩展：纠正错别字，生成多个相似查询提高召回。 |
| F4.4 | 多轮对话中进行指代消解，关联历史上下文。 |

### 步骤5：RAG精准检索
| 需求编号 | 功能描述 |
|----------|----------|
| F5.1 | 向量相似度检索召回Top-K块（阈值可配置）。 |
| F5.2 | 支持混合检索（向量+BM25），提升专有名词匹配。 |
| F5.3 | 使用Cross-Encoder对候选块重排序，压缩至最终N个上下文。 |
| F5.4 | 去重：基于文本哈希或语义相似度去重。 |
| F5.5 | 校验相关性，低于阈值的块丢弃。 |
| F5.6 | 无合格片段时，系统必须拒绝回答并给出明确提示，不得生成无依据内容。 |

### 步骤6：约束生成与溯源标注
| 需求编号 | 功能描述 |
|----------|----------|
| F6.1 | 将精排文本块作为唯一上下文，Prompt强制要求“仅根据上下文回答”。 |
| F6.2 | 答案结构化：核心解答+详细解析+关键知识点枚举，LaTeX正常渲染。 |
| F6.3 | 每个答案段落必须标注来源：[文档名] 第X页，段落Y，置信度Z%。 |
| F6.4 | 记录引用块ID，前端可点击跳转至原文高亮位置。 |

### 步骤7：结果查看与交互
| 需求编号 | 功能描述 |
|----------|----------|
| F7.1 | 答案面板与原文溯源区支持分屏/上下屏显示。 |
| F7.2 | 溯源链接点击后精确定位PDF/文本段落并高亮，支持关键词高亮。 |
| F7.3 | 对答案中的概念自动生成悬浮解释卡片（内部知识库定义优先）。 |
| F7.4 | 支持收藏整条问答，对原文或答案添加划线、批注，批注可导出。 |

### 步骤8：自动生成学习大纲与知识框架
| 需求编号 | 功能描述 |
|----------|----------|
| F8.1 | 单文档一键生成思维导图式大纲，可编辑节点、补充备注。 |
| F8.2 | 支持跨多文档生成综合知识框架，自动去重合并主题。 |
| F8.3 | 基于提问频率自动标注高权重知识点（星标/色条）。 |
| F8.4 | 导出格式：PDF、Markdown、XMind、Anki卡片。 |

### 步骤9：学习行为追踪与薄弱点分析
| 需求编号 | 功能描述 |
|----------|----------|
| F9.1 | 记录搜索、点击溯源、收藏、批注、标记掌握度、答题反馈等行为日志。 |
| F9.2 | 生成个人学习时间线，按文档/知识点维度回溯。 |
| F9.3 | 计算知识点掌握度评分，输出知识图谱雷达图，红色标记薄弱区域。 |
| F9.4 | 支持按时间周期动态调整薄弱点权重。 |

### 步骤10：掌握度标记与智能推送
| 需求编号 | 功能描述 |
|----------|----------|
| F10.1 | 用户可手动标记知识点为：未掌握/模糊/已掌握，或通过答题后自评记录。 |
| F10.2 | 基于遗忘曲线和薄弱点，登录时推送“今日需巩固的N个知识点”卡片。 |
| F10.3 | 可一键生成“专项复习包”，包含知识详解、对比辨析、例题与错题回顾。 |

### 步骤11：搜题、错题归集与备考计划
| 需求编号 | 功能描述 |
|----------|----------|
| F11.1 | 搜题模式支持拍照搜题、题干文字搜索，匹配知识库内试题。 |
| F11.2 | 错题自动归集至个人错题本，记录原题、错误答案、正解、解析、来源。 |
| F11.3 | 基于错题知识点推送同类题目，形成针对性训练卷。 |
| F11.4 | 输入考试日期和目标分数，系统拆解每日学习任务，动态调整备考日历。 |

### 步骤12：管理员审核与优化
| 需求编号 | 功能描述 |
|----------|----------|
| F12.1 | 公共知识库文档审核台：通过/驳回/标注原因，敏感词过滤。 |
| F12.2 | 自动检测内容高度重复文档，提示合并或删除。 |
| F12.3 | 配置部门/班级知识库可见范围，黑名单管理。 |
| F12.4 | 手动触发索引重建，查看检索延迟、召回率抽样，可调整分块与检索阈值。 |

### 步骤13：学习数据统计与多端服务
| 需求编号 | 功能描述 |
|----------|----------|
| F13.1 | 个人仪表盘：学习时长、提问数、收藏数、掌握知识点数、进步趋势。 |
| F13.2 | 可视化图表：每日活跃热力图、掌握进度环图、薄弱点变化曲线。 |
| F13.3 | 对话历史按时间/关键词搜索，可删除、批量导出（PDF/JSON）。 |
| F13.4 | 一键导出学习笔记、错题本、知识大纲，支持包含溯源链接。 |
| F13.5 | 多端实时同步（Web/App/小程序），离线缓存文档与笔记，联网后自动同步。 |

## 3. 非功能需求（摘要）
- 检索响应时间：95%的语义查询在500ms内完成召回。
- 准确率：RAG片段召回率≥90%，Top-3命中率≥85%。
- 无幻觉约束：系统强制不得在无匹配上下文时生成答案。
- 安全性：数据传输加密，知识库私有隔离，权限控制粒度为文档级。
- 可扩展性：支持水平扩展Embedding与向量数据库节点。
